# Employee Management System (Django REST Framework)

## Overview

This project implements an **Employee Management backend system** using **Django REST Framework (DRF)**.  
It provides full **CRUD (Create, Read, Update, Delete)** functionality for managing employee records and follows a **layered architecture** to keep the codebase clean, scalable, and easy to maintain.

---

## Objective

The objective of this project is to implement RESTful APIs for managing employees with proper separation of concerns using:

- Django REST Framework
- Explicit URL mappings
- Layered architecture (Views, Serializers, Services, Managers, Models)

---

## Features

The system supports the following operations:

- Create Employee (POST)
- List Employees (GET)
- Retrieve Employee by ID (GET)
- Update Employee – Full Update (PUT)
- Update Employee – Partial Update (PATCH)
- Delete Employee (DELETE)
